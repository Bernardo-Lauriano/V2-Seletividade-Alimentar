# 🍽️ O que eu quero comer?

Um joguinho interativo e divertido para escolher o que comer! Arraste alimentos para um prato e veja como fica sua refeição.

## 🎮 Como Jogar

1. **Selecione os alimentos** tocando/clicando nos ícones na grade superior
2. **Veja aparecer no prato** a sua composição de refeição
3. **Clique no alimento no prato** para remover
4. **Use o botão "Limpar prato"** para recomeçar

## 🌈 Características

- ✨ **20 alimentos diferentes** para escolher
- 🎨 **Design colorido e atraente** com bordas arco-íris
- 📱 **Responsivo** - funciona em PC, tablet e celular
- 🔄 **Sem pré-requisitos** - funciona direto no navegador
- 🚀 **Rápido** - totalmente hospedado no cliente

## 🛠️ Tecnologia

- **HTML5** - Estrutura
- **CSS3** - Estilo e responsividade
- **JavaScript Vanilla** - Lógica e interatividade
- **Twemoji** - Ícones de alimentos

## 📂 Arquivos

- `index.html` - Página principal
- `styles.css` - Estilos do projeto
- `script.js` - Lógica do jogo

## 🚀 Como usar

1. Clone o repositório ou baixe os arquivos
2. Abra `index.html` no seu navegador
3. Divirta-se! 🎉

## 💡 Dicas

- Tente criar composições balanceadas com proteína, carboidrato e verduras
- O prato muda de tamanho conforme você adiciona mais itens
- Hover sobre os alimentos para ver o efeito visual
